Adds features for DSGVO / GDPR.

== Description ==
Adds features for DSGVO / GDPR, such as:
- Settings page to enter Cookiebot Key
- Once Key is added, automatically adds Cookiebot `script`, appends the key as a `data` attribute, and sets it as `sync`
- Creates new forms & pages for GSGVO/GDPR information requests
- Updates existing Datenschutz / Privacy Policy pages with enhanced security information, including links to above pages/forms and Cookie Information panel
- Instructions on manual steps that should be taken


== Installation ==

= Requirements =
* WordPress Multisite 3.0+
* PHP 5.3*, newer PHP versions will work faster.

= Installation =
* Use the installer via back-end of your install or ...

1. Unpack the download-package
2. Upload the files to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Network/Plugins' menu in WordPress and hit 'Network Activate'
4. No options, no settings, just works
